Answers

1. Please describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.

In HTML, a "parent" is an element that have other elements in it. The ones that are inside are the "child". It is essential to know that because the "parent" decide how the "child" are placed or organized on the website.

2. What are the two main differences between CSS Grid and CSS Flex?

"CSS Grid" is two dimensional, meaning that you can rearrange an element in both row and column while with "CSS Flex" you can only moves it in one line, either row or column.
