Answers

1. When do you use a table?

You use a table when you need to show data in rows and columns, like a schedule, a price list, or any info that is in a grid.

2. When do you not use a table?

You can't use a table when you're making simple lists or layouts that don't need rows and columns.

3. When do you need to do extra work for table accessibility?

You will need to do extra work for table accessibility when you want the readers to understand the table. You can add things like "caption", "th" for headers, and "scope" to tell what each header means.
