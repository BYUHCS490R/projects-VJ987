1. What is JavaScript? How did it get its name?

Java is a programming language used mainly to make web pages interactive.It was originally called "Mocha", then "LiveScript", and finally renamed "JavaScript" for marketing reasons to ride on Java's popularity.

2. What are the two ways you can run JavaScript?

First, you can run it in a browser (with HTML), in the DOM. Second, you can run it with "Node.js" (Backend).

3. What was the most popular front-end js library?

jQuery was the most popular library for many years.

4. What is the most popular front-end js library today?

Today, "REACT" is the most popular JavaScript library.
