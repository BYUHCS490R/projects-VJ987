2. What is the root? What is the relationship between folder and file names and the path?

The root is basically the starting point of a website. In a computer or website, it's like the main folder that holds everything. The path is like the address of a file or a folder. So, when you want to find something, the path tells you where it is and if you want to go to a specific file, the path shows you which folders to open.

3. What is a default document?

A default document is the webpage that shows up automatically when you go to a website or folder without specifying a file.

4. Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link is a full address, and is mainly used to link other website while a relative link is shorter and just gives the path from where you are right now and it is used when you're linking to pages within the same site because it's easier.
