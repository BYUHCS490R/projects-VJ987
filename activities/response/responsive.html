Answers

1. Why is the viewport meta tag important?

It helps the website to look good on different type of screen. Without it, the page will look either too small or too big.

2. How can you adjust the CSS for printing?

By using another CSS file for printing. In that file you can delete part of the page you don't want to print.

3. When would you want to adjust the CSS for printing?

When you want to print a webpage, to make the page look clean and easy to read on paper.

Answers (10/27/2025)

1. Why do we need a viewport meta tag?

The "meta" tag is used to make the website look good on phones and tablets, so that the page fits the screen.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

When you do that, the last one will be used. For example, if one says "column" and the next says "row", the "row" one (the last) wins.

How do you change the CSS for a web page displayed on a phone?

You use a media query (rule) to change how the page looks on a phone.
