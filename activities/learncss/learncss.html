Begining CSS

1. Explain the difference between inline, embedded, and external CSS.

Inline CSS
Used inside the "HTML" tag and it changes only that one thing.
Embedded CSS
Used inside a "style" tag in the HTML file and it changes the whole page.
External CSS
This style is in another "css" file, you then connect that "css" file to you current project so it can change multiple pages.

2. Explain why you should only use external CSS.

"External" is better because all style are in one file, keeping the HTML file clean, making the changes easier

3. What is a selector used for?

A "selector" is used to choose on which HTML element you want to apply the style to
