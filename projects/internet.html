1. Internet

A network that connect computerts from all around the world

2. Web

A collection of websites you can access using the internet

3. How are the Internet and Web different?

Based on the previous answers, the INTERNET is a network infrastructure while the WEB is a service that runs on that infrastructure.

4. ASCII and Unicode

ASCII is a encoding standard that turns letters into numbers so computers can understand them, but it only works for English letters. UNICODE is the same thing but for all the different languages in the world.

5. What is UTF-8

It's the same thing as UNICODE but it also includes emojis and other special symbols

6. URL

It's the address of a webpage on the internet

7. Domain Name

It's a web address that identifies a specific website or webpage on the internet.

8. HTTP

It's the protocol used to transfer and display websites

9. HTTPS

A secure version of HTTP that protect privacy and security

10. IP Address

A unique set of numbers assigned to each devices on the internet.

11. DNS

I's the system that translate the DOMAIN NAME (human-friendly text) into IP ADDRESS (numbers).

12. W3C Recommendations

It's a guideline that ensure that websites and softwares can work together.

13. XML, XHTML, HTML, HTML5

These are called "Markup Languages" used to define and structure informations using TAGS. XML is to store and transport data, XHTML is used to make web pages more precise, HTML is use to have websites with text, pictures and links and HTML5 is a new version of html with new features like videos, sounds and animations to websites.
